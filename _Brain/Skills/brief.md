# Skill : /brief
## Déclencheur
Commande `/brief` + nom du client ou contexte

## Étapes
1. Lire `1_Projets/Pro/[Client]/context.md` si existant
2. Lire `_Brain/Context/identity.md` pour le positionnement
3. Poser 3-5 questions de cadrage si contexte insuffisant
4. Générer le brief structuré :
   - Contexte client
   - Objectif de la mission
   - Périmètre (ce qui est inclus / exclu)
   - Livrables attendus
   - Délais
   - Budget si connu
5. Sauvegarder dans `1_Projets/Pro/[Client]/brief.md`
6. Mettre à jour l'index du dossier client

## Output
Fichier brief.md créé + confirmation
