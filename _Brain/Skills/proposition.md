# Skill : /propose
## Déclencheur
Commande `/propose` + contexte client

## Étapes
1. Lire `1_Projets/Pro/[Client]/context.md` si existant
2. Lire `_Brain/Context/identity.md` (positionnement, offres)
3. Poser les questions manquantes (budget, deadline, objectif précis)
4. Générer la proposition :
   - Compréhension du besoin
   - Solution proposée
   - Livrables détaillés
   - Planning
   - Investissement
   - Prochaines étapes
5. Sauvegarder dans `1_Projets/Pro/[Client]/proposition-[date].md`
6. Mettre à jour l'index client

## Ton
Direct, professionnel, orienté résultat. Pas de remplissage.
