# Skill : /new-client
## Déclencheur
Commande `/new-client` + nom du client

## Étapes
1. Créer le dossier `1_Projets/Pro/[NomClient]/`
2. Créer les fichiers :
   - `context.md` (template ci-dessous)
   - `active.md` (tâche en cours)
   - `index.md` (index du dossier)
3. Ajouter une ligne dans `1_Projets/Pro/index.md`
4. Ajouter une ligne dans `_INDEX.md` racine (Projets actifs)
5. Confirmer ce qui a été créé
